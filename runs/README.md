# Runs Folder

Generated experiment evidence lives here.

- `dataset_eda\` - before/after dataset EDA reports and charts.
- `ml\` - classical machine learning feature extraction and model results.
- `dl\` - deep learning baselines and YOLO training/evaluation runs.
- `comparisons\` - ML vs DL comparison charts.
- `manual_tests\` - no-UI manual prediction outputs.

Most files in this folder are generated artifacts. Keep the final report evidence, but avoid editing these files by hand.
