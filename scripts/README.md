# Scripts Folder

Command-line project tools:

- `dataset_eda.py` - dataset EDA, invalid label checks, class balance charts, duplicate detection.
- `build_tuned_dataset.py` - creates the cleaned/tuned dataset copy.
- `feature_ml_analysis.py` - handcrafted feature extraction and classical ML baseline.
- `deep_learning_baseline.py` - tiny CNN baseline.
- `compare_ml_dl.py` - ML vs DL comparison report.
- `evaluate.py` - YOLO validation/test quality check.
- `export_model.py` - model export helper.
- `predict_images.py` - manual no-UI image prediction and sorting output.
- `run_final_pipeline.ps1` - reproducible final pipeline command sequence.
